"# Funcionalidade de Login" 
"" 
"Esta funcionalidade simula a tela de login do dashboard EduStream." 
"" 
"## Objetivo" 
"Permitir que usuários acessem o dashboard de medição de software." 
"" 
"## Campos previstos" 
"- E-mail do usuário" 
"- Senha" 
"- Botão de acesso" 
"- Mensagem de erro para credenciais inválidas" 
"" 
"## Fluxo esperado" 
"1. O usuário informa e-mail e senha." 
"2. O sistema valida os dados." 
"3. Em caso de sucesso, o usuário acessa o dashboard." 
"4. Em caso de erro, o sistema exibe uma mensagem de falha."
